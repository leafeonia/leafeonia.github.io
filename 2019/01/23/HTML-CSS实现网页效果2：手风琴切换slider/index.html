HTML+CSS实现网页效果2：手风琴切换slider

继续用HTML和CSS尝试搞事

这次学习的是thecodeplayer上的这个案例 Make an accordian style slider in CSS3

HTML

HTML的结构十分清晰，即一个主div块（类名为accordian），其包含一个有着四个表项的无序表。每个表项中包含一个文字链接和一个图片链接。

初始效果：

美化image_title

将link1等文字放置于图片下方，并增加阴影效果。

阴影效果的设置比较简单，通过background: rgba(0,0,0,0.5)即可

放置在图片下方则涉及对CSS中的定位。CSS中有三种定位方式：

  • 相对定位(relative)：相对于其自身在文档流中的原来位置定位
  • 绝对定位(absolute)：相对于离该元素最近的祖先元素进行定位,如果所有祖先元素都没有开启定位，则相对于浏览器窗口进行定位
  • 固定定位(fixed)：类似于绝对定位,不过永远相对于浏览器窗口进行定位，且不会随滚动条滚动 如何在博客里放毒瘤广告

因为img_title是li标签的子元素，所以为了开启绝对定位，还要为li标签开启定位，否则默认为不开启(即position: static)。添加

即可为li开启定位，且不产生其他副作用

CSS：

水平放置

将最大的div块(class=”accordian”)宽度设置为640px,无序表中每个表项的宽度设置为160px，并开启浮动：float:left

效果如下：

这里有两个问题，第一个是li标签自带的小圆点没有消除，在CSS中为li添加list-style-type: none即可消除。

第二个是最后一幅图片会超出640px的宽度限制。这是由于未定义溢出处理方式导致的。同样为CSS添加属性overflow: hidden;可解决此问题。

添加切换效果

实现切换效果的核心思想在于，子元素的样式会覆盖父元素的样式

所以在鼠标放置在最大的父元素.accordian上时，利用hover伪类，为子元素li设置样式width: 40px，此时所有的li的宽度缩小为40px，但是鼠标放置的li除外，因为对于鼠标放置的li，其也设置了样式width:520px，这一样式的优先级高于从accordian继承的优先级，因此其宽度为520px，而非其他li的40px。

再添加渐变效果：transition: all 0.5s

以及边框阴影: box-shadow: 0 0 25px 10px rgba(0, 0, 0, 0.5),就实现了切换动画。
