Lecture 2 : KNN与线性分类器

KNN

CNN(nearest neighbor classifier):选取训练集中与之距离最近的图像的标签作为分类

距离：L1 distance:曼哈顿距离 L2 distance:欧几里得距离

距离是一个超参数(hyperparameter)

时间与训练集的大小成线性关系

kNN(k-nearest neighbor):选取训练集中k个最相近的图像，对它们的标签进行多数表决

线性分类器

$$f(W,x,b) = Wx + b$$

假定图片是一张32*32的只有RGB3通道的图片，则其像素点可以组合为一个3072*1的向量x(3072 = 32*32*3)，而矩阵W的大小为10*3072(10为标签中的类别数),由$W_{10*3072}x_{3072*1} + b_{10*1}=y_{10*1}$计算得到一个10维向量，每一维都代表在某一类中的分数。

Lecture 3 : 损失函数

$$L = \frac{1}{N}\sum_{i=1}^{N}L_i+\lambda R(W)$$

$L$: 总损失

$L_i$:每个测试数据的损失

$\lambda$:正则化强度(regularization strength)，超参数

note:这里的正则化和对数据进行归一化的normalization没有关系

Regularization 正则化

L2正则化（最常用）：$\sum_k\sum_lW_{K,l}^2$

L1正则化：$\sum_k\sum_l|W_{K,l}|$

Elastic Net(L1+L2): $\sum_k\sum_l\beta W_{K,l}+|W_{K,l}|^2$

直观理解正则化的作用：利用尽可能多的W维度

假设图像转化的向量$x = [1,1,1,1]$,类A对应矩阵W中某一行$W_i$，$W_i = [1,0,0,0]$和$W_i = [0.25,0.25,0.25,0.25]$的情况下，该图像在类A的得分都为$W_ix^T = 1$，但$[1,0,0,0]$的正则化损失为1，$[0.25,0.25,0.25,0.25]$的正则化损失为0.25,因此选用后者，原因在于其考虑了4个维度上的信息而前者完全忽略了后面三项的信息。

data loss

SVM loss:

$$L_i = \sum_{j\neq y_i}max(0,s_j-s_{y_i}+1)$$

softmax loss:

$$L_i = -log(\frac{e^{s_{y_i}}}{\sum_je^{s_j}})$$

note:softmax也给出了该测试图像属于某类别的概率

Lecture 4 : 反向传播

利用求偏导的链式法则，从后往前推的过程。

如图，最终结果关于x的偏导数$\frac{\partial{f}}{\partial{x}} = \frac{\partial{f}}{\partial{q}}\frac{\partial{q}}{\partial{x}} = z\frac{\partial{(x+y)}}{\partial{x}} = z*1 = z = -4$

其中,z的值需要在前向过程中先计算得出并存储，才能在反向传播中被使用。反向传播计算偏导（梯度）需要反向过程中从下游传来的偏导(梯度)值,此处即$\frac{\partial{f}}{\partial{q}}$的值

计算出x的梯度为-4，即，x增大，f减小。若f为损失函数，则应减小x从而减小损失，这就是优化W的过程。
